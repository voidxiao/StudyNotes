# MarkDown语法

## 1. 标题
- \# 后接标题
- 一级标题#
- 二级标题##
- 三级标题###
- 四级标题####
- 五级标题#####
- 六级标题######

## 2. **字体**
- *斜体*
  - \* 文字 *

- **加粗**
  - ** 文字 **

- ***斜体加粗***
  - \*** 文字 ***

- ~~删除线~~
  - \~~ 文字 ~~

- ~~***斜体加粗删除***~~
  - \~\~\*\*\*斜体加粗删除\*\*\*\~\~

- 可以混合使用

## 3. **列表**

- 有序列表
  - 1\.
  - 2\.
  - 3\.

- 无序列表
  - 段落前端 -号

- 列表注意缩进


## 4. **链接**
- 内嵌式链接

  1\. 外部链接

     [百度](http://www.baidu.com "打开百度")

      [百度](http://www.baidu.com "打开百度")

  2\.内部链接

     [MarkDown](MarkDown语法.md#MarkDown语法 "回到MarkDown")

	  [MarkDown](MarkDown语法.md#MarkDown语法 "回到MarkDown")

- 引用式链接

  1\. 外部链接

     [百度]

  2\. 内部链接

  [MarkDown]

<!---    链接     -->
[百度]: http://www.baidu.com "打开百度"
[MarkDown]: MarkDown语法.md#MarkDown语法 "回到MarkDown"

	<!---    引用式链接     -->
	[百度]: http://www.baidu.com "打开百度"
	[MarkDown]: MarkDown语法.md#MarkDown语法 "回到MarkDown"

## 5. **图片**
- 内嵌式
  - 外部图片


  ![img](https://www.baidu.com/img/bd_logo1.png?where=super "百度图标")
  ```
  ![img](https://www.baidu.com/img/bd_logo1.png?where=super "百度图标")
  ```
  - 内部图片

  `![img](文件相对路径 "图片描述")`

- 引用式
  - 外部图片

  ![][img]
  ```
  ![][img]
  ```
  - 内部图片

  `![][内部]`

```
<!--- 引用式图片 --->
  [img]: https://www.baidu.com/img/bd_logo1.png?where=super "百度图标"
  [内部]: 内部图片文件相对路径 "图片描述"
```

<!--- 引用式图片 --->
  [img]: https://www.baidu.com/img/bd_logo1.png?where=super "百度图标"

## 6. 引用
- >引用符号 \>
- >>>多重引用 \>>>

## 7. 插入代码
- 在这插入`CODE`代码 符号\`
```
文字 `CODE` 文字
```
- 代码块 符号\```
```
    ```
    CODE 上下两行```包裹
    ```
```
- 设置语法高亮显示
  - 只能代码块使用
    - 在上端\```后面加上所用的语言


```
    ``` 语法高亮语言
    CODE
    ```
```


## 8. 水平分割线
- 有三种方法
  - 第一种`---` 三条横杠
  ---
  - 第二种`***` 三个星号
  ***
  - 第三种`___` 三条下划线
  ___

## 9. HTML语法
- MarkDown支持HTML语法
- MarkDown实现不了可以使用HTML

## 10. 表格
- 展示

  |   这    |    是    |    表    |     格     |
  |-------  |--------  |--------  |---------- |
  | 文字内容 | 文字内容 | 文字内容  |  文字内容  |
  |    这   |    是    |    内    |     容     |

  ```
  |   这    |    是    |    表    |     格     |
  |-------  |--------  |--------  |---------- |
  | 文字内容 | 文字内容 | 文字内容  |  文字内容  |
  |    这   |    是    |    内    |     容     |
  ```
  - 表格可以混合强调语法(字体)
  - 表格的第二行的横杠必须不少于3条,剩下的可以用空格补充

- 左对齐----居中----右对齐

  |   这    |    是    |    表    |     格     |
  |:------- |:--------:|:--------:|----------:|
  | 文字内容 | 文字内容 | 文字内容  |  文字内容  |
  |    这   |    是    |    内    |     容     |

  ```
  |   这    |    是    |    表    |     格     |
  |:------- |:--------:|:--------:|----------:|
  | 文字内容 | 文字内容 | 文字内容  |  文字内容  |
  |    这   |    是    |    内    |     容     |
  ```

  - 符号`:`
  - 横杠左边`:------` 左对齐
  - 横杠两边`:-----:` 居中
  - 横杠右边`------:` 右对齐

# 某些注意点
- 某些符号后面要先加空格,再加文字
- 语句与语句之间最好隔开一行,避免某些奇怪的冲突
- 每行结束打两个空格避免与下一行产生产生问题
